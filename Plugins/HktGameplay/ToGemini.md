공용 api는 HktService에 있다.
HktSimulation 모듈에 작업해야 한다.

HktSimulation은 서버, 클라 공용으로 자연어를 시뮬레이션하는 VM을 처리하는 모듈이다.
이러한 VM을 만들건데 중요한 요소들이다.
1. 명령어( 무엇을 )
2. 태그( 어떻게 )
3. 분기( 언제 )
4. 레지스터( 휘발성, 최적화 )
5. 속성( 영구적 )

가장 중요한 목적은 "인간이 사용하는 자연어와 동일한 수준의 추상화 구현과 동시에 GAS와는 비교도 안되는 궁극의 DOD 형태 최적화"
